This repo uses JJ (Jujutsu) for version control, not git directly. Use `jj` commands for all VCS operations.

Common JJ commands:
- `jj st` — status
- `jj log` — commit log
- `jj diff` — show changes
- `jj new` — create new change
- `jj commit -m "msg"` — commit with message
- `jj describe -m "msg"` — set description on current change
- `jj bookmark set <name>` — set a bookmark (like a branch)
- `jj git push` — push to remote
